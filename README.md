# Prueba_IA

Mini proyecto web con una pantalla de cotizaciones en **peso argentino (ARS)** para:
- Dólar (USD/ARS)
- Bitcoin (BTC/ARS)
- Ethereum (ETH/ARS)

## Ejecutar localmente
```bash
python3 -m http.server 4173 --directory .
```

Abrir en navegador:
- `http://localhost:4173/index.html`

## Proyecto extra: simulador de inversión inteligente
Se agregó un segundo HTML sin reemplazar lo existente:
- `simulador_inversion_inteligente.html`

Abrir en:
- `http://localhost:4173/simulador_inversion_inteligente.html`
